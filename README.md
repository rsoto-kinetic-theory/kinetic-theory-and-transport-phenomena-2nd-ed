# kinetic-theory-and-transport-phenomena-2nd-ed
Exercise codes for:  R. Soto, Kinetic Theory and Transport Phenomena, 2nd Edition, Oxford University Press 2027

All codes written by Rodrigo Soto, Physics Department, Universidad de Chile, rsoto@uchile.cl

Coding style is focus on readbility and direct illustration of the physical ingredients.
No special effort has been put in efficiency. Some possible improvements (not implemented) are commented in the codes.

Comments, suggestions, and detection of errors are welcomed.
